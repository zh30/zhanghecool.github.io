## 我们先举个例子

开发中我们总会有这样的场景：

- 使用 git 进行代码管理

  ```git
  git add --all
  ```

  如果我说我这么写也有相同的效果

  ```git
  gaa
  ```

- 使用 git-flow 初始化项目

  ```git
  git flow init
  ```

  如果我说我这么写也有相同的效果

  ```git
  gfli
  ```
